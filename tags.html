Etiquetas HTML importantes

Esta página está diseñada como una referencia rápida para algunas de las etiquetas comunes cubierto en Aprenda HTML suficiente para ser peligroso . En el proceso de hacerlo, vamos a aprenda cómo hacer tablas HTML a través de table y etiquetas relacionadas.

Las tablas a continuación no incluyen todas las etiquetas HTML, pero enumeran muchas de las más importantes.

Elementos de bloque

Etiqueta Nombre Propósito
h1 – h6 Encabezados Incluye un encabezados (niveles 1 – 6).
p Párrafo Incluye un párrafo de texto.
table Tabla Incluye una tabla.
tr Fila de datos Incluye una fila de datos.
th Encabezado de datos Crea una encabezado de tabla.
td Datos de la tabla Incluye una celda de datos de la tabla.
ol Lista ordenada Incluye una lista ordenada por números.
ul Lista desordenada Incluye lista con viñetas.

Elementos en línea

Etiqueta Nombre Propósito Ejemplo Resultado
em Enfatizado Hacer texto enfatizado <em>sofisticación técnica</em> sofisticación técnica
strong Fuerte Hacer el texto fuerte <strong>al menos mil millones de personas</strong> al menos mil millones de personas
a Anchor Hacer hipervínculo <a href="https://learnenough.com/">Aprenda lo suficiente</a> Aprenda lo suficiente
img Imagen Incluye una imagen <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
header Header Puede contener algunos elementos de encabezado. <header> ... </header>
div Division Agrupa contenidosen bloque <div> ... </div>
span Abarcar Agrupa elementos en línea <span> ... </span>
li Elemento de lista Incluye item de la lista <li>Marineros salados</li>
